We will proceed with "Milestones" until we reach the first stable release. Then we will just switch to a normal versioning system using odd and even numbers to distinguish unstable and stable release.

Initial Development :
  • Milestone 1 (summer 2003)
    • Basic keystrokes
    • Insert/Replace/Command mode
    • Basic commands for editing
    • GUI/CUI Interface API
  • Milestone 2 (end 2003)
    • Plugin system : syntax highlighting
    • Visual mode
    • Markers, Folding, Completion
  • Milestone 3 (middle 2004, maybe longer)
    • Basic scripting support
    • Configuration files
  • Milestone (TBD)
    • TBD ...
    • Vim compatibility layer (read .vimrc files)
    • Remote editing
    • Session transfer
